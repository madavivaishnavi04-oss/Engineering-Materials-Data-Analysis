# Engineering Materials Data Analysis

## Overview
This project presents a comparative analysis of 20 commonly used engineering materials using Microsoft Excel. Mechanical properties including density, ultimate tensile strength (UTS), hardness, yield strength, and Young's modulus were analyzed through charts and a dashboard.

## Objectives
- Compare engineering materials based on mechanical properties.
- Visualize data using Excel charts.
- Develop a dashboard for quick engineering insights.
- Support preliminary material selection.

## Tools Used
- Microsoft Excel
- Microsoft Word
- GitHub

## Key Features
- Dataset of 20 engineering materials
- Five comparison charts
- Interactive dashboard
- Key engineering insights
